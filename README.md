# Gameplay Idea Brainstormer (GIB)

## Contributor
- Yizhi Zhou
- Lincoln Lin

## Brief
This application aims to provide a place for game designers to express, share, store, and edit their gameplay ideas. It can be applied in multiple scenarios. For example, a game company deploys this application to the game design department, enabling all the designers to view, share, and refine ideas. The login process secures safe from data breaches.

## Function
- Login Interface and Validation
  - User needs to log in to access the application
    - Default user name: zyz
    - Default Password: 123456
  - User can also register their own account, and their data will be stored in the database
- Main Interface, CRUD (Create, Read, Update, Delete)
  - User is able to view, create, edit, and delete ideas from the SQL DB
  - User can add customized categories if they want, and delete existing categories
