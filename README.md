# Gameplay Idea Brainstormer (GIB)

## Contributor
- Yizhi Zhou
- Lincoln Lin

## Brief
This application aims to provide a place for game designers to express, share, store, and edit their gameplay ideas.

## Function
- Login Interface and Validation
  - Scenario 1: a game company deploys this application to the game design department, enabling all the designers to view, share and refine ideas. The login process secures safe from data breaches.
- Main Interface, CRUD (Create, Read, Update, Delete)
